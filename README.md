 How It Works

The user enters a text prompt in the frontend UI.

The React application sends the prompt to the Spring Boot backend using REST APIs.

The backend processes the request and communicates with the Stability AI API.

The generated image is returned from the AI service.

The frontend displays the generated image in real time.
